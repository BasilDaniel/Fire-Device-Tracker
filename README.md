# Fire Device Tracker

Мобильное веб-приложение для отслеживания печати и наклейки адресов пожарных устройств.

CSV-колонки: Block, Address, Device ID, Device type, Description, Printed, Installed.
Address: формат `C1.02.040`.

Локальный запуск:

    python3 -m http.server 8080

Откройте http://localhost:8080

Для телефона разместите файлы на статическом хостинге. Отметки сохраняются локально в браузере. Для переноса используйте резервную копию JSON.


## QR-сканер
Кнопка «📷 QR» открывает камеру и подставляет считанное значение в поиск.
Если QR содержит строку с десятизначным Device ID, приложение извлекает этот ID автоматически.
Камера в браузере обычно требует HTTPS или localhost.
